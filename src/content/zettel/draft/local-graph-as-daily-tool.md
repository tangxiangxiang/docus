# 本地图谱

**本地图谱(Local Graph)** 是笔记软件中的一种可视化视图:以**当前正在查看的节点为中心**,只展示它 N 跳(N 通常为 1,有时到 2)以内的邻居节点,以及这些邻居之间的相互链接。Obsidian 笔记页右侧的小型图谱窗格是这种视图的典型实现,Logseq 的"Graph"侧边栏也属于这一类。

特点:

- 节点数 = 当前节点的 1-hop(或 N-hop)邻域,远小于语料库总量
- 通常嵌入在笔记阅读界面里,作为阅读时的上下文导航
- 实现上比 [[global-graph-as-decoration|全局图谱]] 轻量:邻接查询 + 简单几何布局(SVG 或 canvas 均可),不需要完整的力导向仿真

对照:[[global-graph-as-decoration|全局图谱]] 把整个语料库画在一张图上,关注的是"整体结构";本地图谱只画当前笔记附近,关注的是"这篇的上下文"。
