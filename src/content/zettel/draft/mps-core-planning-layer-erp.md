# MPS 作为 ERP 的核心计划层

MPS(Master Production Schedule,主生产计划)是 ERP 系统的核心计划层。它处于高层计划与详细执行层之间,明确规定最终产品需要生产什么、生产多少、什么时候生产。MPS 驱动下游的 [[mrp-material-needs-engine|MRP]] 计算,实际上决定了何时生产、生产多少,因此其准确性直接决定了 MRP 输出是否可行。正如原始笔记所说,MPS 负责"确定最终产品的生产数量和时间"。
