# 思维导图示例

使用 ` ```markmap ` 代码块创建交互式思维导图，支持缩放、拖拽、折叠节点。

```markmap
# 思维导图

## 学习笔记
- 英语学习
  - 语法
  - 词汇
- 技术学习
  - VitePress
  - Vue 3

## 待办事项
- 短期任务
  - 完成文档整理
  - 更新笔记
- 长期目标
  - 建立知识体系

## 工具链
- Claude Code
- Docker
- CodeServer
```
