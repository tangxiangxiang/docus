# Mac 快捷键大全

## 截图

### 系统自带

- `Command + Shift + 3`：截取整个屏幕，保存为 PNG 到桌面
- `Command + Shift + 4`：拖动选择区域截屏，保存到桌面
- `Command + Shift + 5`：打开截屏工具栏（截图/录屏选项）
- `Command + Shift + 6`：截取 Touch Bar

### 截图后编辑

- 按住 `Space`（在选择区域时）：移动选区
- 按住 `Shift`（在选择区域时）：限制为水平/垂直方向
- 按住 `Option`：以点击位置为中心缩放选区
- 按住 `Command` 并拖动：复制选区内容到剪贴板

## Finder 文件管理

- `Command + C`：复制
- `Command + V`：粘贴
- `Command + X`：剪切
- `Command + Z`：撤销
- `Command + Shift + Z`：重做
- `Command + A`：全选
- `Command + D`：复制文件/文件夹（副本在原位置）
- `Command + I`：显示简介
- `Command + R`：显示原身
- `Command + W`：关闭当前窗口
- `Command + Q`：退出应用
- `Command + H`：隐藏当前应用窗口
- `Command + Option + H`：隐藏其他应用窗口
- `Command + N`：新建 Finder 窗口
- `Command + T`：在当前窗口新建标签页
- `Command + ,`：打开偏好设置
- `Delete`：删除到废纸篓
- `Command + Shift + Delete`：清空废纸篓
- `Command + Option + Delete`：直接删除（不经过废纸篓）

---

## 切换与调度

- `Command + Tab`：切换到下一个最近使用的应用
- `Command + Shift + Tab`：切换到上一个应用
- `Command + ~`：在当前应用中切换窗口
- `Command + Option + Esc`：强制退出应用选择器
- `Command + Option + Shift + Esc`（长按）：直接退出当前应用
- `Control + 上/下`：切换全屏模式下同一应用的多个 Spaces
- `Control + 左右`：切换上一个/下一个 Space

---

## 文字编辑

- `Command + B`：粗体
- `Command + I`：斜体
- `Command + U`：下划线
- `Command + K`：添加链接
- `Command + T`：字体面板
- `Command + [`：减少缩进
- `Command + ]`：增加缩进
- `Command + \`：移出文本格式
- `Control + A`：移到行首
- `Control + E`：移到行尾
- `Control + K`：删除到行尾
- `Option + Delete`：删除上一个单词
- `Control + N`：下一行
- `Control + P`：上一行
- `Control + F`：光标前进
- `Control + B`：光标后退
- `Control + H`：删除光标前一个字符
- `Control + D`：删除光标后一个字符

## Safari / 浏览器

- `Command + L`：定位到地址栏
- `Command + Return`：在新标签页打开地址（不跳转到新标签）
- `Command + W`：关闭当前标签
- `Command + T`：打开新标签
- `Command + Shift + T`：重新打开最近关闭的标签
- `Command + N`：新建窗口
- `Command + Shift + N`：新建私密浏览窗口
- `Command + .`：停止加载
- `Command + [`：后退
- `Command + ]`：前进
- `Command + Option + 方向键`：与 `Command + [` / `]` 相同但更高效
- `Command + 数字键`：跳转到指定编号的标签

## 终端

- `Control + C`：取消当前命令
- `Control + Z`：暂停当前命令（可 `fg` 恢复）
- `Control + D`：退出当前 Shell（EOF）
- `Control + L`：清屏
- `Command + K`：清屏
- `Command + T`：新建标签页
- `Command + W`：关闭标签页
- `Command + N`：新建窗口
- `Control + A`：移到行首
- `Control + E`：移到行尾
- `Control + U`：删除整行
- `Command + R`：清空当前行（刷新提示符）

---

## 常用系统快捷键

- `Command + Space`：调出 Spotlight 搜索
- `Command + Option + Space`：调出 Spotlight（文件搜索模式）
- `Control + Space`：切换输入法（需在设置中配置）
- `Command + Option + D`：显示/隐藏程序坞
- `Command + Option + Control + Power`：退出 Mac（无确认对话框）
- `Command + Power`：进入睡眠
- `Control + Power`：显示关机/重启对话框
- `Command + Control + Power`：重启 Mac
- `Command + Control + Option + Power`：立即睡眠

---

## 截图保存位置设置

截图文件默认保存到桌面，如需修改保存位置：

1. 打开「截屏」工具（`Command + Shift + 5`）
2. 点击「选项」→「存储到」→ 选择目标位置
3. 可选择：桌面、文稿、剪贴板、邮件、信息等

## 触控板手势（补充）

- 双指右边缘滑入：打开通知中心
- 双指左边缘滑入：打开 Launchpad
- 三指上滑：显示调度中心
- 三指下滑：显示当前应用所有窗口
- 四指捏合：回到桌面
- 四指展开：从桌面回到上一个应用
